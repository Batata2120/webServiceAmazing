Webservice - Loja de Produtos

Método
URL
Descrição
Parâmetros
Retorno
GET
/cliente
Retorna a lista de todos os clientes cadastrados.
Retorno: [{ "id": 1, "nome": "Nome do Cliente", "email": " email@email.com " }, {...}]
GET
/cliente/:id
Retorna um cliente em específico.
Parâmetros: id (inteiro)
Retorno: { "id": 1, "nome": "Nome do Cliente", "email": " email@email.com " }
POST
/cliente
Insire um novo cliente.
Parâmetros: id (inteiro), nome (string), email (string)
Retorno: { "message": "Cliente inserido com sucesso!" }
PUT
/cliente/:id
Atualiza um cliente existente.
Parâmetros: id (inteiro), nome (string), email (string)
Retorno: { "message": "Cliente atualizado com sucesso!" }
DELETE
/cliente/:id
Exclui um cliente existente.
Parâmetros: id (inteiro)
Retorno: { "message": "Cliente excluído com sucesso!" }
GET
/produto
Retorna a lista de todos os produtos.
Retorno: [{ "id": 1, "nome": "Nome do Produto", "preco": 99.99 }, {...}]
GET
/produto/:id
Retorna um produto em específico.
Parâmetros: id (inteiro)
Retorno: { "id": 1, "nome": "Nome do Produto", "preco": 99.99 }
POST
/produto
Insere um novo produto.
Parâmetros: id (inteiro), nome (string), preco (float)
Retorno: { "message": "Produto inserido com sucesso!" }
PUT
/produto/:id
Atualiza um produto existente.
Parâmetros: id (inteiro), nome (string), preco (float)
Retorno: { "message": "Produto atualizado com sucesso!" }
DELETE
/produto/:id
Exclui um produto existente.
Parâmetros: id (inteiro)
Retorno: { "message": "Produto excluído com sucesso!" }
GET
/compra
Retorna a lista de todas as compras cadastradas.
Retorno: [{ "id": 1, "data": "2021-05-01", "total": 99,99 }, {...}]
GET
/compra/:id
Retorna uma compra específica.
Parâmetros: id (inteiro)
Retorno: { "id": 1, "data": "2021-05-01", "total": 99,99 }
POST
/compra
Insere uma nova compra.
Parâmetros: id (inteiro), dados (string), total (float)
Retorno: { "message": "Compra inserida com sucesso!" }
PUT
/compra/:id
Atualiza uma compra existente.
Parâmetros: id (inteiro), data (string), total (float)
Retorno: { "message": "Compra atualizada com sucesso!" }
DELETE
/compra/:id
Exclui uma compra existente.
Parâmetros: id (inteiro)
Retorno: { "message": "Compra arquivo com sucesso!" }
GET
/produtocompra
Retorna a lista de produtos e compras.
Retorno: [{ "id_produto": 1, "id_compra": 1, "quantidade": 1 }, {...}]
GET
/produtocompra/:id_produto/:id_compra
Retorna a quantidade de um produto específico em uma compra específica.
Parâmetros: id_produto (inteiro), id_compra (inteiro)
Retorno: { "quantidade": 1 }
POST
/produtocompra
Insere um novo produto na compra.
Parâmetros: id_produto (inteiro), id_compra (inteiro), quantidade (inteiro)
Retorno: { "message": "Produto adicionado à compra com sucesso!" }
PUT
/produtocompra/:id_produto/:id_compra
Atualiza a quantidade de um produto específico de uma compra específica.
Parâmetros: id_produto (inteiro), id_compra (inteiro), quantidade (inteiro)
Retorno: { "message": "Quantidade do produto na compra atualizada com sucesso!" }
DELETE
/produtocompra/:id_produto/:id_compra
Exclui um produto da compra.
Parâmetros: id_produto (inteiro), id_compra (inteiro)
Retorno: { "message": "Produto removido da compra com sucesso!" }
